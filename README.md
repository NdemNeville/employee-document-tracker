# Employee Document Tracker

Employee Document Tracker is a Django-based web application designed to help organizations manage employee records and their required documents.

## Features

* Add, edit, and delete employees
* Organize employees by department:

  * Front Office (FO)
  * Field Maintenance Engineers (FME)
  * Back Office (BO)
* Upload and manage employee documents
* Track document status
* Search employees to view their documents

## Technologies Used

* Python
* Django
* HTML
* SQLite

## Installation

1. Clone the repository

git clone https://github.com/NdemNeville/employee-document-tracker.git

2. Navigate into the project

cd employee-document-tracker

3. Install dependencies

pip install -r requirements.txt

4. Run migrations

python manage.py migrate

5. Start the server

python manage.py runserver
